1. 메뉴 바 만들기 
  1.1 베리언트 예제에 왼쪽 상단에 pages +를 클릭해서 page를 추가한다. 
  1.2 #로 프레임을 1400x1000이상으로 만든다. 
      어느정도 드레그하고 오른쪽에서 너비를 1400. 높이를 1000으로 설정한다. 
  1.3 사각형 툴 ㅁ 로 너비 360 높이 56인 사각형을 그린후 fill #000000으로 설정한다.
   
2. 메뉴 아이콘 만들기 
  2.1 Ctrl + 를 눌러 화면을 800% 이상으로 확대한다.  
  2.2 사각형 툴 ㅁ 로  18*2 인 직사각형을 만들고 복제해서 3개 만든다.
  2.3 사각형 툴 ㅁ 로  3개의 사각형을 감싸는 
      24*24 인 직사각형을 하나 더 만들고 Fill 메뉴에서 투명도를 0으로 설정한다.
  2.4 4개의 사각형을 모두 선택하고 그룹 Ctrl + G 를 눌러 그룹으로 만든다.    
  2.5 group의 색상는 #fffffff로 변경한다. 
  2.6 Shift +1을 누르면 확대되었던 화면이 딱 맞게 축소된다. 
  2.7 Group1를 선택하여  검은 사각형 왼쪽에 왼쪽 간격 16을 두고 배치한다. 
      Group1을 선택한 상태에서 Alt 를 누르면서 Object로 마우스를 가져가면 간격을 볼수 있다 
      모두 16으로  미세한 조정을 방향키로 한다. 
      
3. 태블릿 메뉴 바 만들기 
  3.1 Alt + 드레그로 복제하고 너비를 600으로 변경한다.
  
4. 데스크톱 메뉴바 만들기         
  4.1 사각형만 복제해서 너비를 1280으로 설정한다.
  4.2 텍스트 툴(T)를 이용해서 [서비스 메뉴 01]이라고 입력후 왼쪽 여백을 140으로 설정한다.  
  4.3 사각형을 1*14 #ffffff로 만들어 서비스 메뉴 01 옆에 둔다.
  4.4 문자와 흰색라인을 선택하고 Shift + A를 눌러 오토레이아웃 적용 후 사이 간격을 24로 설정한다.  
  4.5 레이어를 열어 문자를 선택하고 Ctrl + D를 눌러 복제하면 간경를 유지한 채 복제가된다. 5개를 더 복제한다. 
  4.6 Rectangle 6(문자옆에 흰색 선)을 선택하고 Ctrl + X를 눌러 잘라낸다. 
  4.7 왼쪽 레이어에서 첫번째 문자를 선택하고 Ctrl + V 를 누르면 문자 뒤로 선이 배치된다.
  4.8 각각의 문자를 선택 후  Ctrl + V 를 한다. 
  4.9 메뉴바의 글자를 [서비스 메뉴 01]~[서비스 메뉴 06]으로 변경한다. 
  
5. 이름 지정하기 
  5.1 첫번째 메뉴바를 드레그해서 Ctrl+G로 그룹을 만들고 이름을 Mobile로 한다. 
  5.2 두번째 메뉴바를 드레그해서 Ctrl+G로 그룹을 만들고 이름을 Tablet로 한다.
  5.3 세번째 메뉴바를 드레그해서 Ctrl+G로 그룹을 만들고 이름을 Desktop로 한다.  
  
6. 베리언트 설정하기 
  6.1 3개의 메뉴바를 드레그로 선택 후 Frame 옆 ◈V 눌러 Create Component Set을 클릭
  6.2 오른쪽 상단에 [Properties] 메뉴 옵션을 Mode로 변경한다. 
  6.3 Assets에서   < 눌러 뒤로 돌아간 후 page2를 선택해 component를 드레그하고 mode를 변경해 본다.
