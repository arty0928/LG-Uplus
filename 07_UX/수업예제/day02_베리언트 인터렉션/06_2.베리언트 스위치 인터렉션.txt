1. 스위치 버튼 만들기
  1.1 #을 선택하고 우측 화면에서 iPhone 16를 선택한다.
  1.2 사각형 툴(□)로 120 * 60인 직사각형을 만들고 
  	  radios를 30으로 설정하여 둥근 버튼형대로 만든다. 
  1.3 원형 툴로 54*54로 만들고 색상은 #ffffff 좌,상하 여백을 3으로 지정한다. 
      원형을 선택한 상태로 Alt를 클릭하고 버튼에 마우스를 over하면 간격을 볼 수 있다.
      
2. shadow 지정하기  
  2.1 원을 선택하고 오른쪽 패널 하단에 [Effects] 메뉴의 + 를 클릭하고 
      Drop shadow가 나타나면 왼쪽 □를 클릭한다. 
  2.2 옵션값을 'X 2' 'Y 2' 'Blur 2' 투명도(Color옆에 %)는 25%로 설정한다.
  2.3 버튼을 선택하고 오른쪽 패널 하단에 [Effects] 메뉴의 + 를 클릭하고 
      Drop shadow가 나타나면 왼쪽 □를 클릭한다.
  2.4 옵션값을 'X 4' 'Y 4' 'Blur 4' 투명도(Color옆에 %)는 40%로 설정한다.
     
3. 베리언트 만들기
  3.1 모두 선택하여 Alt+드레그하여 복제한다. 
  3.2 모두 선택된 상태에서 우측 패널의 Aligment 우측 정렬을 선택한다.   
  3.3 원을 선택해서 왼쪽 방향키 <-를 3번 눌러 오른쪽 여백을 3으로 맞춘다.
  3.4 버튼의 색을 #FF004D로 변경한다. 
  3.5 Ctrl+G로 각각를 group화 한다.
  3.6 두개의 버튼 그룹을 선택하고 우측 패널에서 ◈ V 를 누르고
      Create Component set을 선택한다.   
      
4. 인터렉션 설정하기 
  4.1 우측 패널에서 Prototype 메뉴를 선택하고 왼쪽 회색 버튼을 더블 클릭해서 선택한다. 
      옵션을 기본으로
  4.2 선 위에 마우스를 올려놓고 + 에서부터 드레그해서 빨간 버튼에 연결한다.
  	  옵션을 기본으로
  4.3 오른쪽 버튼도 더블 클릭하여 + 에서부터 드레그해서 왼쪽에 연결한다.
  	  옵션을 기본으로	  
  4.4 베리언트(컴포넌트)이름을 Switch로 변경한다. 
  
5. 인터렉션 컴포넌트 만들기 
  5.1 왼쪽 페널에서 +를 눌러 새로운 페이지를 추가한다. 
  5.2 아래 패널에서 #을 눌러 새로운 프레임을 선택후 우측 패널에서 iPhone을 선택한다. 
  5.3 Assets에서 Switch를 프레임 내부로 드레그해서 배치한다. 
  5.4 아무것도 선택하지 않은 상태에서 우측 패널에서 [Prototype] 메뉴를 선택하고 
  	  원하는 디바이스를 설정하고 프로토 타입을 ▷ 실행한다.
