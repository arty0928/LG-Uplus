1. 복제하기 
   제목(컨스트레이트 예제) 옆에 v 를 누른 후 Duplicate to your drafts를 선택하면
   내 프로젝트에 복제가된다.
   1.2 좌측 페널에서 Frame1 더블 클릭해서 이름을 Background로 변경한다. 
2. 오토레이아웃
   2.1 사각형 툴(ㅁ, 단축키 R) 선택후 드래그하고 오른쪽 패널에서 너비 50, 높이 50으로 
       색상을 변경한다.
   2.2 사각형을 Alt+드래그로 복제한다. 
   2.3 두 사각형을 드래그해서 선택하고 우측 패널에서 Layout 라인에있는
       Use Auto Layout 버튼을 클릭하거나 단축키 Shift+A를 눌러 오토레이아웃으로 만든다. 
       ==> Frame2가 새로 생기고 두 사각형이 Frame2에 들어갔다. 
   2.4 간격 조절하기 
       우측 페널에 Auto Layout 옵션에 ][로 두 사각형의 간격을 설정하거나 
       두 사각형 사이를 마우스로 클릭하면 ][이 나타나고 Hover 상태에서 마우스를 이동하면 간격이 변한다. 
       간격을 20으로 변경한다. 
   2.5 패딩값 조절하기 
       padding 옵션이 default는 좌우, 상하이며 옆 □ 을 선택하면 상하좌우의 패딩 값을 설정할 수 있다.
       
3. 최소 최대 영역 설정하기 
  3.1 우측 사각형을 더블클릭하거나 왼쪽 페널에서 선택하고 Ctrl+C, Ctrl+V, Ctrl+V 해서 복제하거나 
      Ctrl+D로 복제하면 간격을 유지하면서 복제된다.
  3.2 Frame1를 선택하고 우측 패널에서 W v를 선택해서 add min width를 120와 
      add max width 600으로 지정하고 Frame1의 크기를 변경해보면 최대와 최소사이서 움직인다. 
  3.3 Alignment를 중으로 변경하면 frame1의 위치가 변경된다. 
  3.4 clip content를 선택하면 줄어든 frame의 크기에 맞춰 사각형들이 숨겨져서 보이지 않는다. 
  3.5 Direction을 ↓로 변경하면 오토 레이아웃의 방향을 수직 정렬로 변경한다. 

4. 프레임 넓이 조절하기 
  4.1 Frame1을 선택고 w:500,h:300으로 변경하면 Fixed로 변경된다. 
  4.2 Resize to Fit 를 클릭하면 패딩은 그대로 둔체 내부 오브젝트 크기에 딱 맞게 줄어들고
      가로 세로가 Hug로 변경된다. 
  4.3 direction옵션에 wrap을 클릭한 후 frame1의 너비를 줄이면 사각형이 프레밈의 크기에 맞게 배치된다.  

5. 버튼에 오토 레이아웃 적용하기
  5.1 Frame1은 좌측 페널에서 눈감기를 선택해 잠시 가려둔다. 
  5.2 상단에 있느 버튼을 선택하고 Alt+드래그로 복제한다.
  5.3 선택된 버튼을 shift+A로 오토레이아웃을 만든다.    
  5.4 Frmae2가 선택된 상태에서 우측 패널의 W를 Hug로 변경하고 버튼 텍스트를 변경하면 프레임 크기가 같이 변경된다.  
  
6. 버튼에 아이콘 insert하기 
  6.1 상단에 있는 작은 하트 아이콘을 Alt+드래그로 복제한뒤 버튼 컨테이너 내부로 드레그하면 
      파란 막대가 생겨난다.  큰 하트는 컨테이너 보다 커서 내부로 안 넣어진다. 
  6.2 간격을 16으로 변경한다. 
  
7. 큰 아이콘 inset하기 
  7.1 버튼을 Alt+드래그로 복제하고 Shift+A로 Auto layout을 만든다. 
  7.2 상단의 큰 아이콘을 Alt+드래그로 복제한다. 
  7.3 큰 아이콘을 드래그해서 버튼 내부에 넣을 때 Ctrl을 누르면 파란선 나타난다.
  7.4 Resize to Fit 를 클릭하면 컨테이너에 컴포넌트 크기에 맞게된다. 
    
8. 스마트 그리드(오브젝트 내부에 점이 생겨 간격이나 위치를 쉽게 조절할 수 있는 상태) 
  8.1 하단에 프레임(#, 단축키 F)를 선택하고 드래그해서 300*56 크기 fill메뉴에서 색상을 선택한다. 
  8.2 상단의 작은 아이콘들을 모두 선택하고 Alt+드래그해서 복제하고 새로운 프레임 안에 배치한다. 
  8.3 좌측 페널에서 Icons들을 Shift로 선택하고 ctrl+r 을 누르렴 이름을 변경하는 창이 뜬다. 
  8.4 Rename to 부분에 Icon을 입력하고 Numbers↑를 선택한다 
  8.5 Frame4를 선택하고 Frame4에 마우스를 hover하면 내부 아이콘에 동그란 점이 보인다. 
      (Ctrl+마우스휠로 화면을 확대하면 더 잘 보인다.)
  8.6 첫번째 아이콘을 마우스로 hover하면 점이 커진다. 이때 드레그해서 원하는 위치로 변경할 수 있다. 
      => Frame4 내부의 아이콘 순서는 변하지 않는다. 
  8.7 아이콘 사이의 분홍색 바를 드래그 하면 아이콘의 간격을 변경할 수 있다. 
  
9. Auto layout 적용하기
  9.1 Frame4을 선택하고 Alt+드래그하여 복제하고  shift+A로 Auto layout 상태로 변경한다. 
  9.2 아이콘 사이에 마우스를 hover하여 분홍색 바를 드레그 하면 간격을 변경할 수 있다. 
  9.3 아이콘을 더블 클릭하거나 좌측 패널에서 아이콘을 선택후 마우스로 드래그하면 위치가 변경된다. 
      ==> 아이콘의 순서가 변경된다
  9.4 아이콘을 선택한 상태에서 좌우 방향키로 순서를 변경할 수 있다.
