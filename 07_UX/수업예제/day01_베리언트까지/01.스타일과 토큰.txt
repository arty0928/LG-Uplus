1. 새 프레임 만들기 
   하단에 프레임 툴(#, 단축키 F)을 선택하고 드레그 하거나 우측 패널에서 원하는 크기의 프레임을 선택한다. 
   
   
2. 도형 그리기 
   하단에 사각형 툴 (□, 단축키 R)을 클릭하면 사각형을 생성하고  사각형 툴 옆에(v)를 선택하면 
   원형(단축키 O), 다각형, 화살표(단축키 Shift+L), 라인(단축키 L)등을 선택 후 드레그 해본다. 
   
3. 스타일 추가하기 
   빈 화면을 선택하면  우측 페널에 Local styles + 를 선택하면 Text, Color, Effect, Grid를 추가할 수 있다. 
   3.1 Color 추가하기 
     Local styles + 에서 Color를 선택하고 Name에 색상을  Properties에 색상 값을 입력하거나
     색상을 선택하여 원하는 색상을 선택후 Create sytle 버튼을 클릭한다. 
   3.2 색상 적용하기
     사각형을 그리면 우측 페널에 Fill 항목에 기본값이 나온다. 
     fill 오른쪽의 style버튼을 선택해서 등록한 color를 선택한다.
      
   3.3 다른 색상 적용하기
     선택한 색상 오른쪽으로 - 버튼을 누르면 선택한 색상이 사라진다. 
     다시 sytle 버튼을 선택해서 원하는 색샹을 선택한다.
    
   3.4 색상 스타일 변경하기 
     빈 화면을 선택하면  우측 페널에 Local styles 메뉴에 등록한 색상에 마우스를 over하면
     우측에 edit style 버튼이 생긴다. 클릭해서 새로운 값으로 설정할 수 있다. 
     
   3.5 그룹명 지정하고 색상 추가하기 
   그룹명/이름  75p 참고해서 색상을 등록해 보기 
   
4. 텍스트 스타일 만들고 수정하기
   하단 프레임에서 텍스트 툴(T)로 프레임 내부를 클릭하여 문자를 입력한다. 
   우측 페널에서 Text 메뉴에서 폰트의 종류와 크기 두께 등를 변경한다. 
     
5. 디자인 토큰 사용하기 
  5.1 하단의 리소스 툴을 선택하고 plugins&widgets을 선택하고 
      search에 Token을 입력하면 Token studio for Figma가 나온다. 
      run 버튼을 클릭해서 플로그 인을 설치한다.
       
  5.2 하단의 리소스 툴에서 설치된 Token studio for Figma를 선택한다. 
      리소스툴 단축키(ctrl+p)
  5.3 new empty file을 선택한다. 등록하는 방법은 local과 같다. 
  
  5.4 local style 불러오기 
      하단의 sytles & Variables를 클릭하고 import styles를 선택한다. 
      원하는 스타일을 선택 후 import > import all
      
  5.5 스타일을 local로 내보내기 
  	  하단의 sytles & Variables를 클릭하고 Export styels & variables to Figma를 선택한다. 
  	  원하는 스타일들을 선택하고 confirm 버튼을 클릭한다. 
  
  5.6 {}를 클릭해서 만든 토큰의 json을 확인한다. 
  
  5.7 배포한 tokens.json의 값을 복사한다. 
  
  5.8 github에 연동하기(배포한 파일 확인하기)
